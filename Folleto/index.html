¿Qué es HTML?

Html es un lenguaje muy sencillo que permite escribir hipertexto es decir texto presentado de forma estructurada y agradable con enlaces que conducen a otros documentos o fuentes de información .


La descripción se basa en especificar en el texto la estructura lógica del contenido (títulos, párrafos de texto normal, enumeraciones, definiciones, citas etc) así como los diferentes efectos que se quieren dar ( especificar los lugares del documento donde se debe poner cursiva, negrita o un gráfico determinado) y dejar que luego la presentación final de dicho hipertexto se realice por un programa especializado (como un navegador de internet)


Las etiquetas están formadas por determinados códigos metidos entre los signos y con la barra / cuando se trata de la segunda etiqueta en infecto (la de cierre) por ejemplo para abrir y cerrar ciertas directivas solo se pone una vez en el lugar del texto donde queremos Que aparezca el efecto.




Estructura básica de un documento de Html .

Un documento html comienza con la etiqueta < html >y termina con la etiqueta < / html > Sirve para indicar el inicio y el final de la estructura de una página en html.

El encabezamiento delimitado por < head > y < / head > que sirven para definir un valores válidos en todo el documento.

y sirve para indicar el inicio y el final del título de la página que aparecerá en el navegador de internet.

El cuerpo delimitado por < body > y < / body > Donde reside la información del documento. Es decir es la parte donde se escribe todo el contenido de la página como textos, enlaces, las tablas, imágenes, etcétera.

La estructura básica de un documento html queda de la siguiente forma:

< HTML >

< HEAD >

< TITLE > < / TITLE >

< / HEAD >

< BODY >

< / BODY >

< / HTML >
