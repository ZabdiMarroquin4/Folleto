FINAL.

Para finalizar con html podemos decir que html es un markup lenguaje, lo que significa que está escrito con códigos qué puede leer una persona sin que se dé necesario copiar lo primero.

En otras palabras el texto de una página web está marcado con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto estás etiquetas de marcado son las propias etiquetas de html.

Cuando escribes código en HTML estás escribiendo etiquetas HTML todas las etiquetas HTML cumplen con la gente características:

• El carácter del menor que.

• El carácter mayor que.

• Una palabra o carácter que determina que etiquetas se está descubriendo.

• Algunas etiquetas solicitan uno o más números de atributos y esto debe contar con un "valor".

• La gran mayoría de etiquetas cuentan con etiqueta de apertura y etiqueta de cierre.
