Código para agregar imágenes.

< HTML >

< HEAD >

< TITLE>imágenes< / TITLE >

< / HEAD >

< BODY >

< IMG SRC="DESCARGA.JPG" HEIGHT="200" WIDTH="450"

< / BODY >

< / HTML >

Se usa la etiqueta < IMG SRC > más el nombre de cómo hayamos guardado la imagen y se le agrega . jpg dentro de comillas dobles. HEIGHT es para indicar el alto y WIDTH es para indicar el ancho.
